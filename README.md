# Grocery App (Dev Mode with JSON)
This is a grocery shop mobile app using Python + Kivy. For now, it uses local `.json` files.

## Requirements
- Python 3.10+
- Kivy (`pip install kivy`)

## Run the App
```bash
python main.py
```

## Build APK (later)
Install buildozer and run:
```bash
buildozer init
buildozer -v android debug
```
